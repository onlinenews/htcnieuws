'Microsoft wil dat HTC Windows Phone op haar Android-smartphones zet'

Microsoft heeft aan HTC gevraagd of het wellicht Windows Phone mee kan leveren op haar Android-smartphones. De software reus wil op deze manier haar marktaandeel met het OS vergroten.

Het nieuws is afkomstig van Bloomberg. De bron heeft van meerdere personen binnen de twee bedrijven vernomen dat Microsoft voorgesteld heeft aan HTC om Windows Phone mee te leveren als systeem op diens Android-smartphones. Onduidelijk is of Microsoft wil dat consumenten kunnen kiezen welk systeem ze opstarten op de telefoon, of dat in plaats van Android standaard Windows Phone op de smartphones wordt mee geleverd van HTC.

Microsoft zou in ruil hiervoor hebben aangeboden om de licentiekosten voor haar systeem met een flinke korting te verstrekken aan HTC. Ook zouden andere bronnen hebben gezegd dat Microsoft zelf bereid was om haar licentiekosten helemaal op te heffen voor HTC om zo een extra stimulans aan het bedrijf te geven om Windows Phone mee te leveren op haar ontwikkelde Android-smartphones. De gesprekken zouden nog wel in een vroeg stadium verkeren, aldus Bloomberg.

Door Windows Phone op de Android-smartphones mee te leveren zou HTC geen nieuwe smartphones hoeven te ontwikkelen. De fabrikant zou hierdoor veel kosten besparen en alsnog een nieuwe telefoon in de markt kunnen zetten. Ook zou een dualboot-optie een concurrentie voordeel kunnen opleveren voor de noodlijdende fabrikant.
