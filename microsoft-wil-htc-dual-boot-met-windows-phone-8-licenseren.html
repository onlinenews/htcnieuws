Microsoft wil HTC dual-boot met Windows Phone 8 licenseren

4 oktober om 13:49

Een wel heel opvallend nieuwtje. Microsoft probeert HTC over te halen naast Android op hun telefoons ook Windows Phone 8 te installeren. De toekomstige telefoons van HTC zouden dan kunnen opstarten met zowel Windows Phone 8 als met Android, naar keuze van de gebruiker. Het idee voor deze constructie zou afkomstig zijn van Microsoft, die zelfs bereid is de licentiekosten voor Windows Phone 8 terug te brengen om HTC over de streep te trekken.

Geen Windows Phone 8 meer van HTC

Deze strategische beslissing komt niet op een vreemd moment. Diverse bronnen claimden al eerder dat HTC van plan was te stoppen met Windows Phone-toestellen, omdat Microsoft recent de toesteldivisie van Nokia heeft overgenomen. Het Taiwanese bedrijf zou daardoor geen heil meer zien in het uitbrengen van telefoons met WIndows Phone 8. Het plan om twee besturingssystemen op een toestel te installeren kan daarmee worden getypeerd als de gulden middenweg. Een gulden middenweg waar voornamelijk Microsoft van profiteert, want Windows Phone heeft momenteel een verwaarloosbaar marktaandeel. Of de combinatie er gaat komen, is nog onzeker. Mocht zo'n toestel ooit het levenslicht zien, dan lees je dat uiteraard hier op GSMinfo.nl.
